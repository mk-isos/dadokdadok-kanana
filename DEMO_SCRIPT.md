# 다독다독 × Kanana-o 데모 촬영 스크립트

목적: 2~4분 내에 핵심 흐름(책 선택 → 듣기 → 말하기 → 리포트)을 명확히 보여주기

## 1) API Key 입력
- 상단 `API Key 설정`을 열고 키를 입력합니다.
- "키가 브라우저(localStorage)에만 저장된다"는 점을 짧게 설명합니다.

## 2) 책 선택
- Step 1에서 3권 중 한 권을 선택합니다.
- 감정 태그(예: 걱정, 기쁨, 슬픔)에 따라 대화 분위기가 달라진다는 점을 보여줍니다.

## 3) 그림책 보기
- 선택한 책의 이미지와 원문을 보여줍니다.
- `이야기 살펴보기` 버튼으로 Kanana 분석 결과가 들어오는 장면을 확인합니다.

## 4) 아이 수준에 맞게 문장 바꾸기
- 기초/표준/심화 중 하나를 선택하고 `아이에게 맞게 바꾸기`를 누릅니다.
- 변환된 문장을 적용해 다음 듣기 단계에서 바로 쓰이는 점을 보여줍니다.

## 5) 이야기 듣기
- `원문 듣기`와 `쉬운 글 듣기`를 각각 1회씩 실행합니다.
- Kanana TTS 우선, 실패 시 브라우저 음성 fallback으로 이어지는 안정성을 설명합니다.

## 6) Kanana와 말하기
- 마이크 버튼으로 1~2턴 대화를 진행합니다.
- 아이 답변 → Kanana 피드백 → 다음 질문으로 이어지는 멀티턴 구조를 보여줍니다.

## 7) 보호자 리포트 확인
- `오늘 이야기 정리하기` 버튼을 눌러 리포트를 생성합니다.
- "오늘 장면, 아이 강점, 다음 질문"이 한눈에 정리되는 결과를 보여주고 마무리합니다.

---

## 촬영 팁
- 한 장면당 10~20초 내외로 짧게 유지합니다.
- 버튼 클릭 후 로딩 상태와 결과 카드가 바뀌는 순간을 꼭 담습니다.
- 마지막에 안전 문구(진단/치료 목적 아님)를 화면에서 확인하고 종료합니다.
